# FoxScript
 A fun little programming langauge I am currently working on. It started as a random experiment in language, ocmpiler, and VM design. FoxScript may not be deisgned for production use, but either does or will support
 all primitive types, functions, recursion, control flow (if/else, loops, etc), static typing, basic terminal IO, and static scope checking. 

 # instalation
 clone the repo and use the make command to compile.

 # usage
 run "./FoxScript -f "your file path"" or "./FoxScript -f "Your file Path" -d" for debugging info like token stream and AST print out.
 See testing .fox files for exmaple code and syntax. Alternatively look thorugh the grammar defenition doc for syntax. Proper documentation will be writeen soon.
